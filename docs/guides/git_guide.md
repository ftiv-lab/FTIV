## 🛠️ 基本的な「戻し方」コマンド

### 1. 「作業中の変更」をすべて捨てて、安定版に戻したい
```bash
# 全ファイルの変更を破棄して、v1.0.2 のクリーンな状態へ戻る（注意：取り消せません）
git reset --hard v1.0.2
```

### 2. 現在の作業を「一旦保存」してから別のことをしたい
```bash
# 1. 開発中ブランチを作成して保存
git checkout -b feat/my-new-feature
git add .
git commit -m "wip: working on something"

# 2. 安全な基盤に戻る
git checkout master
```
