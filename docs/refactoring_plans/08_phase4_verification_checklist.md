# Phase 4: 手動検証チェックリスト

今回のリファクタリング（コネクタロジックの移動）に伴い、以下の機能が正しく動作するかをご確認ください。

## 前提条件
1. アプリを起動する。
2. テキストウィンドウまたは画像ウィンドウを2つ以上作成する。
3. それらを接続（Connect）して、画面上にコネクタ（線）がある状態にする。
4. **Sceneタブ > Connections サブタブ** を開いておく。

## チェック項目

### 1. 選択と表示の同期
- [ ] **コネクタの選択**: 画面上のコネクタ（線）をクリックする。
    - **確認**: Connectionsタブの「Selected」欄に `Connector (...)` と表示されるか？
    - **確認**: 「Delete Line」「Line Color」などのボタンが有効化（クリック可能）になるか？
- [ ] **選択解除**: 画面の何もないところをクリックする。
    - **確認**: 「Selected」欄が `None` (または未選択表示) に戻るか？
    - **確認**: ボタンが無効化（グレーアウト）されるか？

### 2. 個別操作 (Selected Actions)
コネクタを選択した状態で実施してください。

- [ ] **削除 (Delete)**: 「Delete Line」ボタンを押す。
    - **確認**: 線が削除されるか？
    - **確認**: 削除後、UIの選択状態が解除されるか？
- [ ] **色変更 (Color)**: 「Line Color」ボタンで色を変更する。
    - **確認**: 線の色が即座に変わるか？
- [ ] **太さ変更 (Width)**: 「Line Width」ボタンで太さを変える。
    - **確認**: 線の太さが変わり、プレビューと確定が動作するか？
- [ ] **不透明度 (Opacity)**: 「Line Opacity」ボタンで変更する。
    - **確認**: 線の透明度が変わるか？

### 3. 矢印とラベル (Arrows & Labels)
- [ ] **矢印スタイル**: Start / End / Both / None ボタンを押す。
    - **確認**: 線の先端に矢印が付く/消えるか？
    - **確認**: 別のコネクタを選択してから戻ったとき、正しいボタンが押下状態になっているか？
- [ ] **ラベル操作**: 「Add Label」または「Toggle Label」ボタンを押す。
    - **確認**: ラベルが表示/非表示になるか？
    - **確認**: ラベル部分をクリックしても、親のコネクタが選択状態になるか？

### 4. 一括操作 (Bulk Actions)
複数のコネクタを作成した状態で実施してください。

- [ ] **一括色変更**: 下部の「All Connectors」エリアの「Line Color」を使用。
    - **確認**: 全てのコネクタの色が変わるか？
- [ ] **一括太さ変更**: 「Line Width」を使用。
    - **確認**: 全てのコネクタの太さが変わるか？

---
何か挙動がおかしい（ボタンが反応しない、エラーが出る等）場合はお知らせください。
