FTIV

Floating Text Image Viewer

デスクトップ上にテキストや画像を自由に配置できるオーバーレイツール

スクリーンショット準備中
assets/screenshot.png (推奨: 1200×675px)

主な機能

  • テキストや画像をデスクトップ上の好きな位置に配置
  • 縁取り・影・グラデーションによるテキスト装飾
  • 縦書き表示に対応
  • APNG / GIF アニメーションの再生
  • ウィンドウ同士をコネクタ線で接続
  • レイヤー管理（親子グループ化・並べ替え）
  • クリックスルー（背面のアプリを操作透過）
  • シーンの保存と復元
  • 不透明度の調整
  • 前面固定（Always on Top）

対応形式

種別 形式
テキスト UTF-8 テキスト（多言語対応）
画像 PNG, JPEG, BMP, GIF, APNG, WebP
シーン .ftiv（独自形式）

動作環境

項目 要件
OS Windows 10 / 11（64bit）
インストール 不要（ポータブル版 / インストーラー版）
