FTIV

Floating Text Image Viewer

A desktop overlay tool that lets you place text and images freely on your screen

Screenshot coming soon
assets/screenshot.png (recommended: 1200x675px)

Features

  • Place text and images anywhere on your desktop
  • Text decoration with outlines, shadows, and gradients
  • Vertical writing support
  • APNG / GIF animation playback
  • Connect windows with connector lines
  • Layer management (parent-child grouping and reordering)
  • Click-through (interact with apps behind overlays)
  • Save and restore scenes
  • Adjustable opacity
  • Always on Top

Supported Formats

Type Formats
Text UTF-8 text (multilingual)
Images PNG, JPEG, BMP, GIF, APNG, WebP
Scenes .ftiv (proprietary format)
